# Microcystis Meta-analysis

This repository contains the R scripts used in the analyses accompanying the manuscript:

Temperature and Nitrate Effects on Biomass Accumulation and Microcystin Compartmentalization in Microcystis spp.: A Multilevel Meta-analysis.

The repository contains scripts for:

• Data preparation

• Multilevel meta-analysis

• Forest plots

• Funnel plots

• Publication bias analyses

• Summary figures
